Email Signature Builder

Create a professional email signature for your company in minutes.

Gmail (web)
  1. Click the gear icon in the top right → See all settings.
  2. On the General tab, scroll to Signature.
  3. Click Create new, enter a name, and paste the copied signature into the editor.
  4. Under Signature defaults, select your new signature for new emails and replies.
  5. Scroll to the bottom and click Save Changes.
Outlook (Microsoft 365 / new Outlook for Windows & Mac)
  1. Open Outlook and go to Settings (gear icon) → Accounts → Signatures.
  2. Click + New signature and give it a name.
  3. Paste the copied signature into the editor.
  4. Under Select default signatures, assign it to New messages and Replies/forwards.
  5. Click Save.
If icons or the logo look broken, make sure "Download external images" is enabled in Outlook's privacy settings.
Outlook (classic desktop for Windows)
  1. Go to File → Options → Mail → Signatures…
  2. Click New, name the signature, and click OK.
  3. Click inside the edit box and paste the copied signature.
  4. On the right, choose the signature for New messages and Replies/forwards.
  5. Click OK to save.
Classic Outlook uses Word's rendering engine and may display images slightly differently. Paste using Ctrl+V, not "Keep Text Only".
Outlook on the web (outlook.com / Office 365 webmail)
  1. Click the gear icon in the top right → View all Outlook settings.
  2. Go to Mail → Compose and reply.
  3. Under Email signature, click + New signature, name it, and paste the signature.
  4. Select it as the default for new messages and replies.
  5. Click Save.
Apple Mail (macOS)
  1. Open Mail → Settings… → Signatures.
  2. Select your email account on the left, then click + to add a new signature.
  3. Name the signature and paste the copied block into the right panel.
  4. Important: uncheck "Always match my default message font" so formatting is preserved.
  5. Drag the signature to the account, or pick it from the dropdown when composing.
Front
  1. Click your avatar in the bottom left → Settings.
  2. Go to Signatures (under "Personal preferences" or "Company").
  3. Click Create signature, enter a name, and paste the copied signature into the editor.
  4. Under Channel access, assign it to the channels where it should appear.
  5. Click Save.
Front may resize images based on their natural dimensions. If icons look too large, let IT know.
Yahoo Mail
  1. Click Settings (gear icon) → More Settings.
  2. Go to Writing email.
  3. Toggle on Signature, then paste the copied block into the editor.
  4. The signature saves automatically.
Thunderbird
  1. Go to Account Settings for your email account.
  2. Check Use HTML under Signature text.
  3. Paste the copied signature into the text area.
  4. Click OK to save.
If pasting as plain text, save the signature as an .html file and use "Attach the signature from a file" instead.
Spark / Superhuman / Mimestream / other modern clients
  1. Open the app's Settings or Preferences.
  2. Find the Signatures section.
  3. Create a new signature and paste the copied block into the rich-text editor.
  4. Assign it as the default for your account and save.
