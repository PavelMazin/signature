AEC Email Signature Builder

  1. Fill in your details in the form, including website.
  2. Check the live signature preview.
  3. Click Copy Signature.
  4. Open Gmail and go to Settings → See all settings → General → Signature.
  5. Create a new signature or open an existing one, then paste the copied block.
  6. At the very bottom of the page, click Save Changes.

This page is made for employees. You do not need to edit any HTML code.
